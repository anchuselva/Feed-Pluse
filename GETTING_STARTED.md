# 🚀 Getting Started with FeedPulse Premium

## Quick Start Guide

Welcome to FeedPulse Premium! This guide will help you explore all the features in just 5 minutes.

---

## 📍 You Are Here

The application is **already running** in your browser. No installation needed!

---

## 🎯 Step 1: Submit Your First Feedback (1 minute)

You're currently on the **Public Feedback Form**.

### What to do:
1. **Fill in the Title:**
   ```
   Example: "Add export to PDF feature"
   ```

2. **Write a Description (min 20 characters):**
   ```
   Example: "It would be great if I could export my feedback reports to PDF format for sharing with my team. This would make it much easier to present insights during meetings."
   ```

   👉 Watch the **character counter** and **progress bar** below the description!

3. **Select a Category:**
   - Try: **✨ Feature Request**

4. **Add Your Info (optional):**
   - Name: Your name
   - Email: Your email

5. **Click "Submit Feedback"**
   - Watch the button show: "Processing with AI..."
   - See the **AI analysis happen** (takes ~1.5 seconds)
   - Success message appears!

### What just happened:
- Your feedback was analyzed by AI
- Sentiment was detected
- Priority score calculated (1-10)
- Relevant tags extracted
- Summary generated
- All data saved to localStorage

---

## 🔐 Step 2: Access Admin Dashboard (1 minute)

Now let's see the admin side!

### What to do:
1. **Click "Admin Login →"** at the bottom of the page

2. **You'll see the login page:**
   - Notice the **glassmorphic card** and **animated background**
   - Demo credentials are displayed for you

3. **Enter credentials** (or click to auto-fill):
   ```
   Email: admin@feedpulse.com
   Password: admin123
   ```

4. **Click "Sign In"**
   - Watch the loading animation
   - Success toast appears
   - Automatic redirect to dashboard

### What just happened:
- JWT token generated
- Auth state updated
- Token saved to localStorage
- You're now authenticated!

---

## 📊 Step 3: Explore the Dashboard (2 minutes)

Welcome to the **Premium Dashboard**!

### What you see:

#### Top Bar
- **Dashboard title** and description
- **User profile** badge (your email)
- **Logout button** (try it later!)

#### Sidebar (Left)
- **FeedPulse logo** with gradient icon
- **Navigation tabs:**
  - Dashboard (current)
  - Feedback
  - Analytics
- **Dark mode toggle** at bottom (try it now!)

#### Main Content Area

**1. Stats Cards (4 cards at top):**
- Total Feedback: How many submissions
- Open Items: Unresolved feedback
- Avg Priority: Average AI priority score
- Top Tag: Most common tag

👉 **Hover over cards** to see the shadow effect!

**2. Charts (2 charts):**
- **Left:** Line chart showing 7-day trend
- **Right:** Pie chart showing category distribution

👉 **Hover over charts** to see interactive tooltips!

**3. Feedback List (below charts):**
- Your submitted feedback appears here
- Each card shows:
  - Title + category badge
  - Sentiment badge (color-coded)
  - AI summary
  - Date and priority score
  - AI tags
  - Status dropdown

👉 **Click any card** to see full details in a modal!

---

## 🌓 Step 4: Try Dark Mode (30 seconds)

### What to do:
1. **Look at the sidebar** (left side)
2. **Scroll to the bottom**
3. **Click the sun/moon icon**
4. **Watch the magic happen!**

### What you'll see:
- Entire app transitions smoothly
- Background changes to dark gradients
- Cards become dark with blur
- Text inverts to white
- Icons and badges adjust colors
- Charts update their colors
- Icon rotates during transition!

👉 **Toggle back and forth** to see the smooth transition!

---

## 🔍 Step 5: Use Filters (1 minute)

Let's filter the feedback!

### What to do:
1. **Click "Feedback" in sidebar** (middle icon)
2. **You'll see 3 filter controls:**

   **Search:**
   - Type any keyword (e.g., "export", "pdf")
   - Results filter in real-time

   **Category:**
   - Select "Feature Request"
   - Only features show

   **Status:**
   - Select "New"
   - Only new items show

3. **Notice the active filter badges** below
   - Each filter shows as a removable badge
   - Click the X to remove a filter

4. **Click "Dashboard" or "Analytics"** in sidebar
   - See how the view changes
   - Charts update based on current data
